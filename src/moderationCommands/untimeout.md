# $untimeout

## Description <br>

This command is used to remove timeout from users. Simple.

## Arguements <br>

**1** arguement. <br>
``$untimeout (user)``

### Parameters
- ``User`` ``Type (User | Required)`` Example of Arguement: **@Electro#9999** <br>

### Example with Entered Arguements
- ``$untimeout @Electro#9999`` <br> <br>
